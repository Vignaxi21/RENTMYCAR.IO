About Us

RentMyCar.io is a platform for local residents, and businesses to advertise their vehicles that need to be rented. Our goal is to offer an attractive and functional user interface for car owners to be able to hire out their own cars to generate an additional income from their assets The functions car owners are given below:
• A user is able to register with the system by filling up a registration form.
• A user is able to login to the system by providing a combination of username and password, which was setup by the user during registration.

Once logged in a user should be able to;
o Add details of a vehicle to the system by completing an add vehicle details form.
o View the vehicles added by the user to the system.
o Edit the details of a particular vehicle
o Delete a vehicle from the system.
